# 🚗 Sistema de Gerenciamento para Concessionária

Sistema desktop desenvolvido em **Python** para gerenciamento de uma concessionária de veículos. O projeto foi desenvolvido em equipe como atividade acadêmica, com foco na aplicação de conceitos de programação, banco de dados e desenvolvimento de interfaces gráficas.

## ✨ Funcionalidades

- 🔐 Login e autenticação de usuários
- 👥 Gerenciamento de usuários
- 🛡️ Controle de permissões de acesso
- 🚗 Cadastro e gerenciamento de veículos
- 👤 Cadastro e consulta de clientes
- 👨‍💼 Cadastro e consulta de funcionários
- 📅 Agendamento de atendimentos
- 💰 Registro de vendas
- 🚙 Controle de frota
- 📊 Dashboard com informações do sistema
- 📄 Geração de relatórios em PDF
- 🔎 Pesquisa de clientes e funcionários
- 💾 Banco de dados SQLite

## 🚀 Como executar

1. Clone este repositório:

```bash
git clone https://github.com/dniel-404/car-dealership-system.git
```

2. Instale as dependências do projeto:

```bash
pip install tkcalendar bcrypt reportlab
```
3. Execute o arquivo principal do sistema.
